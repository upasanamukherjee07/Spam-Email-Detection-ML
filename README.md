# Spam-Email-Detection-ML
Intermediate level spam detection system using SVM and TF-IDF


This project implements a machine learning based spam detection system using SVM and TF-IDF feature extraction.


- Text preprocessing using regex
- TF-IDF vectorization
- Support Vector Machine (SVM) classifier
- Confusion matrix evaluation
- Interactive user input system


Achieved ~99% accuracy on SMS Spam Collection dataset.


- Python
- Pandas
- Scikit-learn
- Matplotlib
